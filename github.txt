Pour importer le dossier sur un fichier :
	- Aller sur le dossier
	- Clique droit et github clone
	- Faire ok et mettre le lien du fichier de github(clone and download) et faire clone


Plus précisement :

	- Pour enregistrer un fichier dans github on va d'abord alors sur le dossier où l'on veut
	mettre le lien gitHub. Ensuite on va faire clique droit et getExt clone pour cloner le lien gitHub
	dans son dossier.  
	- Un dossier avec le nom du gitHub a donc été créer. C'est dans ce dossier que l'on va mettre les fichier
	que l'on veut ajouter sur le gitHub. 
	- Pour ajouter le fichier il faut le commit. Pour ce faire on va selectionner le fichier et faire clique
	droit et gitExt commit et faire le commit and push. Après ça le fichier est copié sur le gitHub
